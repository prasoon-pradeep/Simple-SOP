How it works

2

3

Why SOP Builder

Everything you need. Nothing you don't.

Data Sovereignty

All SOPs live in a local SQLite database on your machine. No account required, no data leaves your network.

Structured Editor

Purpose, scope, safety notes, tools, items, and sequential steps — each field has a home. Auto-saves as you type.

Visual Steps

Attach annotated photos to each step. Draw arrows, circles, and labels directly in the app.

PDF Export

Generate a printable, text-selectable PDF from any SOP in one click. No formatting work required.

Portable Bundles

Export any SOP as a self-contained .sop file. Share it offline, import it anywhere SOP Builder is installed.

Revision History

Log every revision with notes, approver, and date. Full audit trail built in, no plugins needed.

AI Text Enhancement

Rewrite any prose field with one click using your own Anthropic, OpenAI, or Gemini API key. Improves clarity without changing your content.

Yes — free for personal use and internal business use, with no time limit. The source code is available on GitHub under an MIT + Commons Clause licence. The Commons Clause means you cannot sell SOP Builder or offer it as a paid hosted service. See the full licence for details.
Yes, fully for core features. Creating, editing, exporting PDFs, and sharing .sop bundles all work with no internet. The only exception is the optional AI text enhancement feature, which contacts the AI provider you configure — but that feature requires you to add your own API key and is never used unless you trigger it explicitly.
All SOPs — including attached images — are stored in a single SQLite database in your operating system's app data directory. On Windows: %APPDATA%\sop-builder. On Linux: ~/.local/share/sop-builder. On macOS: ~/Library/Application Support/sop-builder. Nothing is transmitted or synced anywhere.
Export individual SOPs as .sop bundles from within the app — each bundle is self-contained with all images. You can also copy the entire database file from the app data directory. During the alpha period we recommend regular exports for anything you can't afford to lose.
SOP Builder checks for updates automatically each time you launch the app. If a new version is available, you'll be prompted to install it — no manual download needed. The update is downloaded and applied in the background, and takes effect on the next restart.

Note: an internet connection is required to check for and download updates. This is the only time the app needs network access. Your SOP data is never involved in the update process.
Yes — macOS (Apple Silicon, M1 or later) is supported as of v0.2.0. Download the .dmg from the releases page. On first launch, right-click the app → Open → Open to bypass the Gatekeeper warning (the app is unsigned). Intel Mac support is not available yet.
